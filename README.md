# TomaChocolate
> "Toma chocolate, paga lo que debes."

**TomaChocolate** es una herramienta web diseñada para optimizar el reparto de gastos en reuniones, asados o viajes, eliminando la fricción de los registros de usuario.

### Objetivo
Resolver el cálculo de deudas compartidas minimizando la cantidad de transferencias necesarias entre los usuarios.

### Tech Stack (MVP)
* **Backend:** Java 21 + Spring Boot 4
* **Database:** PostgreSQL (JPA/Hibernate)
* **Doc:** Swagger/OpenAPI
